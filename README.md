## Run Locally 

### Clone the project
```plaintext
git clone https://github.com/JaviNGD/collections-app.git
```

### Go to the project directory
```plaintext
cd collections-app
```

### Install dependencies
```plaintext
  npm install
```

### Start the server
```plaintext
  npm run dev
```

### See page in
```plaintext
  http://localhost:5173/collections-app/
```
